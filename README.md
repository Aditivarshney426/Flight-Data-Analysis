# ✈️ Flight Data Analysis Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing flight data using Power BI to identify patterns in delays, cancellations, and airline performance.

The dashboard provides interactive visual insights for better decision-making.

---

## 📊 Dataset Information
The dataset includes:
- Flight details
- Departure and arrival times
- Delay information
- Airline data
- Airport data

---

## ❓ Key Business Questions Solved
- Which airlines have the highest delays?
- Which routes are most frequently delayed?
- Monthly trend of flight delays
- Cancellation patterns
- On-time vs delayed flight performance

---

## 📊 Dashboard Features
- KPI cards (Total Flights, Delayed Flights, On-Time %)
- Monthly trend analysis
- Airline-wise performance
- Delay distribution charts
- Interactive filters (Month, Airline, Route)

---

## 🛠 Tools Used
- Power BI
- Data Cleaning & Transformation

---

## 📈 Key Learnings
- Data visualization techniques
- Dashboard design
- Data cleaning in Power BI
- Business insights generation
